(constant) AuthProvider

AuthProvider component is a wrapper that provides the authentication context to the children components. It manages the session token state, allowing components to access and update it. - Provides the `sessionToken` and `setSessionToken` via context. - Any component wrapped in this provider can access or modify the authentication state.
